# AI-assisted-Retail-performance-and-Customer-behavior
AI-assisted retail analytics project analyzing 3,900 transactions to uncover customer behaviour and revenue drivers. AI tools (ChatGPT, Julius AI) supported data profiling, cleaning strategies, and query ideas, while Python, PostgreSQL, and Power BI were used for implementation, analysis, and interactive dashboard visualization.
AI ASSISTED DATA SUMMARY:
Before starting the analysis, AI-assisted profiling was used to quickly understand dataset structure, detect potential data quality issues, and recommend cleaning strategies. Using AI tools, the dataset was evaluated for:
•	Missing values
•	Data type inconsistencies
•	Outliers and anomalies
•	Potential feature engineering opportunities
These AI suggestions helped accelerate the exploratory phase and ensured a structured data cleaning approach.
TOOLS USED AND PURPOSE:
Julius AI	- Dataset profiling, identifying missing values, and recommending cleaning strategies
ChatGPT - Assisted with data cleaning logic, SQL query optimization, and insight formulation
Python (Pandas)	- Data cleaning, transformation, and exploratory data analysis
PostgreSQL - Advanced data analysis using SQL queries
Microsoft Power BI	- Interactive dashboard creation and business insight visualization
